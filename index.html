Etiquetas HTML

Tarea 2. Etiquetas HTML: Nombre, Descripcion y Ejemplo


Nombre de la Etiqueta Descripción Ejemplo
<!DOCTYPE html> Define el tipo de documento y la versión de HTML que se esta utilizando. <!DOCTYPE html>
<html> Es la raíz de un documento HTML, y contiene todo el contenido de la página web. <html> ... </html>
<head> Contiene metadatos, enlaces a archivos CSS, scripts y el título de la página. <head> ... </head>
<title> Define el título del documento, que se muestra en la barra de titulo del navegador. <title>Mi Página Web</title>
<body> Contiene todo el contenido visible de la página, como texto, imagenes, etc. <body>Contenido visible</body>
<h1> Define un encabezado de nivel 1, utilizado para títulos importantes. <h1>Encabezado principal</h1>
<h2> Define un encabezado de nivel 2, utilizado para títulos importantes. <h2>Encabezado secundario</h2>
<p> Define un parrafo de texto. <p>Este es un párrafo de ejemplo.</p>
<a> Define un enlace a otra pagina o recurso. <a href="https://www.example.com">Visitar sitio</a>
<img> Define una imagen en el documento. <img src="imagen.jpg" alt="Descripción de la imagen">
<div> Define una division o sección de la página, util para agrupar contenido. <div>Contenido dentro de un div</div>
<td> Esta se utiliza para definir una celda dentro de una tabla. <td>Contenido dentro de una tabla.</td>
<br> Esta se utiliza para saltar una linea en la pagina <td> ... </td>
<b> Define texto en negrita. <b>Texto en negrita</b>
<u> Define texto subrayado. <u>Texto subrayado</u>
<hr> Inserta una linea horizontal. <hr>
<li> Define un item en una lista. <ul><li>Elemento de lista</li></ul>
<table> Define una tabla. <table> ... </table>
<th> Define una celda de encabezado en una tabla. <th>Encabezado de columna</th>
<form> Define un formulario. <form action="procesar.php"> ... </form>
<button> Define un botón para interactuar con el usuario. <button>Haz clic aquí</button>
<option> Define una opción dentro de un menú desplegable. <option>Opción 1</option>
<textarea> Define un área de texto multilinea. <textarea>Texto por defecto</textarea>
<mark> Resalta un texto para marcarlo. <mark>Texto resaltado</mark>
<sub> Define texto en **subíndice**. <sub>H</sub>₂O
